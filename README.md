dna-frequences
==============

#### Technologies:
  - [Angular.js]
  - [Underscore]
  - [Twitter Bootstrap]
  - [Karma]
  - [Grunt]
  - [Bower]
  - [Yeoman]
  - [SCSS]

### Build instructions
  - Install node.js, npm, bower, yeoman, ruby and compass gem
  - Switch to application folder and execute "npm install"
  - For run application need execute "grunt server"
  - For run unit tests need execute "grunt test"
